# flow-state

> 这个工具来自我自己的困扰，也来自三年职业生涯规划咨询中，我听过无数次的同一个问题：**「我怎么才能找到自己真正投入的状态？」**

---

## 为什么我做了这个工具

我做职业生涯规划顾问将近三年，接触过两类人最多。

一类是职场人，他们来找我，通常是因为站在一个岔路口：要不要离职？要不要换方向？要不要创业？这些问题的背后，往往藏着同一件事 —— 他们在现在的工作里，很少感到真正的投入。

另一类是 15～20 岁的年轻人。他们在传统教育体系里感到格格不入，却在 AI 时代找到了某种出口，开始用 AI 把脑子里的想法变成东西，甚至有了创业的念头。但他们面对的最大困惑是：**我到底能做什么？什么是我真正想做的事？**

还有一小部分人，是刚开始创业的人。时间突然多了，反而更难聚焦，很难真正开始做自己的事。

**这最后一类，说的也是我自己。**

在上班的时候，我发现自己几乎只能在深夜才能真正进入状态 —— 那些需要创意、需要思考的工作，白天根本做不了。开会、消息、各种打扰，把注意力切得七零八碎。为了赶进度，我经常熬夜，睡不好，压力很大，陷入一个恶性循环。

去年我正式离开职场，开始创业。我以为时间自由了，状态会更好。结果发现根本不是这样。一个人在家或者在办公室，手机、电脑、各种消息反而更难屏蔽。我甚至给自己设了固定的「上班时间」，强迫自己坐在那里工作 —— 看起来很努力，但其实心不在焉，什么都做不好。

我在咨询中发现，大多数人卡在三个地方：

1. **不太清楚心流到底是什么，也不知道它真正的价值在哪里**
2. **不知道如何找到属于自己的心流** —— 每个人的生活节奏不同，触发心流的条件也完全不同，没有一个通用答案
3. **即使知道方法，也很难坚持下去** —— 这是最普遍的问题，因为大多数方法都太抽象，很难落地

这个工具想解决的，正是这三件事。

**它不会告诉你心流是什么。它会帮你从自己的真实生活里，找到属于你自己的答案。**

无论你是想换一份工作、推进一个重要项目、还是在创业初期找到真正的节奏 —— 那些值得做的事，都需要你进入心流状态才能真正做好。

---

## 它能做什么

每周设置 40 次提醒（每天 5～6 次）。只需运行一次 `/flow-state:flow-setup`，之后电脑会**在设定的时间自动弹出对话框** —— 不需要打开 Claude Code，3 步填完关掉，约 15 秒。不在电脑旁时，在手机上随手记一句话，回到电脑后用 `/flow-state:flow-import` 导入。坚持 1～2 周后，运行 `/flow-state:flow-review` 找到你的规律：

1. **哪些活动会触发心流** — 你在做什么、在哪里、和谁在一起
2. **你的高效时段** — 一天中哪些时刻最容易进入心流
3. **如何重新设计你的日程** — 基于真实数据的可操作建议

**适合所有人使用，建议每隔 3～6 个月重新测一次** —— 生活状态变了，心流的来源也会跟着变。

```bash
/plugin marketplace add Richology/flow-state
/plugin install flow-state@richology-flow-state
```

**没有 Claude Code？** 这套工具同样适用于 ChatGPT、Claude 网页版、Gemini 或任何 AI 工具。查看 [`prompts/`](prompts/) 文件夹，里面有可以直接复制使用的独立提示词，不需要安装任何东西。

## 工作原理

`flow-setup` 会检测你的操作系统，自动选择对应方案：

- **macOS**：在系统 cron 里安装定时任务，到点运行 `remind.sh`，通过 AppleScript 弹出原生对话框
- **Windows**：在任务计划程序（Task Scheduler）里安装定时任务，到点运行 `remind.ps1`，通过 PowerShell Windows Forms 弹出对话框

两个平台的弹窗体验完全一致，数据格式也相同。

```
cron / Task Scheduler → remind.sh / remind.ps1 → 对话框 → ~/.flow-data/logs.json
```

## ⚠️ 必做：系统权限设置

### macOS

**这一步必须完成，否则提醒会静默失败，不弹任何窗口。**

macOS Catalina 及更新版本中，cron 需要「完全磁盘访问权限」才能运行弹窗脚本。

1. 打开「**系统设置 → 隐私与安全性 → 完全磁盘访问权限**」
2. 点击左下角 `+` 按钮
3. 按 `Cmd+Shift+G`，输入 `/usr/sbin/cron`，回车
4. 选中 `cron`，点击「打开」

### Windows

**PowerShell 默认执行策略会阻止脚本运行，必须提前设置。**

以管理员身份运行 PowerShell，执行：

```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

## 两条记录路径

这个工具解决了一个真实的问题：**你不可能一天都坐在电脑前。**

心流可能发生在任何地方 —— 在咖啡馆和朋友聊天时、跑步时、发呆时、做饭时。如果只记录「在用电脑时」的状态，你会错过生活里最重要的一部分数据，分析结论也会有偏差。

所以这个工具设计了两条路径：

**电脑端（自动）**
到点弹窗，**3 步完成，约 15 秒**：
1. 「在做什么，在哪/和谁」—— 一行写完
2. 三个按钮选心流状态：否 / 部分 / 是
3. 「8 7」两个数字填专注和情绪

不打断你，填完继续做事。

**手机端（随手记）**
不在电脑旁时，在手机备忘录、微信或任何地方随手写一句话，比如：
- 「下午在咖啡馆写东西，很投入」
- 「3点 开会 公司 无聊死了」
- 「刚跑完步，脑子很清醒，好久没这种感觉了」

回到电脑后，把这些文字粘贴给 `/flow-state:flow-import`，Claude 会自动解析成结构化数据存入记录。

**关于手机端记录的习惯养成**

不需要一开始就写得完整。初期只要能记下「在做什么、感觉怎么样」就够了。每次导入后，工具会给你一条具体的提示，告诉你下次可以多加什么信息 —— 只说一件事，不会同时给你五条建议。

由简到繁，慢慢来。坚持比完整更重要。

## 五个 Skill

### `/flow-state:flow-setup`
第一次使用时运行。自动检测 macOS 或 Windows，告诉它你的起床和睡觉时间，它会自动：
- 计算 6 个均匀分布的提醒时间点
- 将提醒脚本复制到 `~/.flow-data/`（macOS: `remind.sh`，Windows: `remind.ps1`）
- 安装定时任务（macOS: cron，Windows: Task Scheduler）

### `/flow-state:flow-log`
通过 Claude Code 手动记录 —— 适合在计划外的时刻补录。约 30 秒完成，记录内容：
- 当前在做什么
- 在哪里、和谁在一起
- 专注程度（1～10）
- 情绪状态（1～10）
- 心流状态（是 / 部分 / 否）
- 可选备注

数据保存到 `~/.flow-data/logs.json`。

### `/flow-state:flow-review`
分析你积累的数据。**3 条记录就能运行**，40 条以上效果最佳。

```bash
/flow-state:flow-review        # 分析全部数据
/flow-state:flow-review week   # 只分析最近 7 天
```

分析内容包括：
- 各活动的心流率排名
- 一天中的心流热力图
- 地点与陪伴者的影响分析
- 3 条个性化可操作建议

**三个特别设计：**

- **冷启动模式**：记录少时不拒绝分析，而是展示「早期信号」，找出你目前最突出的心流时刻，让你第一周就有收获
- **历史对比**：每次分析自动保存快照，下次运行时对比变化 —— 心流率有没有提升、高峰时段有没有变化，让你看到自己在进步
- **反思环节**：分析结束后问你一个问题：「有没有让你意外的地方？」你的回答和数据一起保存，下次分析时回顾，帮你看到自己认知的变化

### `/flow-state:flow-import`
把手机上随手记的文字导入进来。一句话也行，多条一起发也行，格式完全不重要。

Claude 会从你的文字里提取时间、活动、地点、陪伴者、专注度、情绪和心流状态，缺失的字段标记为空，不会报错或拒绝。解析完后会展示结果让你确认，有疑问的地方才会问你。

每次导入结束后，只给你一条具体的改进提示，比如：
> 「下次可以加上大概几点，比如『下午3点在咖啡馆写方案，很投入』，这样我能分析你哪个时段最容易进入状态。」

一次只说一件事。下次自然就记得更完整了。

### `/flow-state:flow-uninstall`
移除 `flow-setup` 安装的所有定时提醒（自动识别 macOS/Windows），并询问是否保留历史记录数据。

## 数据存储

所有记录保存在本地的 `~/.flow-data/logs.json` 文件中，不会上传或发送到任何外部服务。

```json
[
  {
    "timestamp": "2026-04-18T14:32:00",
    "activity": "写作",
    "location": "家里",
    "with_whom": "独自一人",
    "focus": 9,
    "mood": 8,
    "flow": "yes",
    "note": ""
  }
]
```

随时可以停止提醒：
```bash
/flow-state:flow-uninstall
```

## 背后的科学原理

Csikszentmihalyi 的研究发现，人们对「什么让自己快乐或专注」的预测往往严重失准。经验采样法通过捕捉**实时快照**绕过了这个问题 —— 你记录的是当下真实的感受，而不是事后的回忆（记忆会美化或扭曲）。

每周 40 个样本，1～2 周内即可积累足够的数据，找到真正属于你的心流规律。
