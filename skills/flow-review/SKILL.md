# 心流规律分析

## 第一步：读取数据

读取 `~/.flow-data/logs.json`。

如果文件不存在，提示用户先运行 `/flow-state:flow-setup` 开始记录。

如果 `$ARGUMENTS` 为 `week`，只分析最近 7 天；否则分析全部数据。

统计记录总数，进入对应模式：

- **3～9 条**：冷启动模式
- **10～39 条**：初步分析模式
- **40 条以上**：完整分析模式

---

## 冷启动模式（3～9 条）

不要说「数据不够，无法分析」。而是说：

```
你已经有 X 条记录了，我来告诉你目前观察到的早期信号。
这还不是最终结论，但已经能看出一些苗头。
```

然后做以下分析：

1. **目前最常出现的活动是什么**，以及这些活动里心流出现了几次
2. **心流比例是多少**（是/部分/否 各几条）
3. **有没有一条记录特别突出**——专注度最高或心流最强的那条，把它单独展示出来，问用户：「这次你在做什么？当时有什么特别的地方吗？」

最后告诉用户：
```
再积累 X 条记录，我就能给你完整的规律分析。
按照现在的频率，大约还需要 X 天。继续记录！
```

---

## 初步分析模式（10～39 条）和完整分析模式（40 条以上）

### 一、数据概览

```
记录总数：X 条（来自提醒弹窗 X 条 + 手动导入 X 条）
时间跨度：X 天
心流分布：是 X% / 部分 X% / 否 X%
平均专注度：X.X / 10
平均情绪值：X.X / 10
```

### 二、哪些活动最容易触发心流？

按活动分组，计算心流率、平均专注度、平均情绪值、次数，按心流率排序：

```
排名  活动        心流率   专注度   情绪   次数
1     [活动]      85%     8.5     8.2    7次
...
```

解读：指出心流率最高的 2～3 个活动的共同特征。

### 三、一天中哪个时段最容易进入心流？

按时段分组（上午/下午/傍晚/夜间），展示心流率热力图：

```
时段         心流率   专注度   记录数
上午 6-12    —%      —       —条
下午 12-18   —%      —       —条
傍晚 18-21   —%      —       —条
夜间 21-24   —%      —       —条
```

解读：指出高峰时段，给出具体的日程安排建议。

### 四、环境与陪伴的影响

分析地点和陪伴者对心流的影响，展示对比数据，指出最有利的环境组合。

### 五、低效时段分析

找出心流率最低的时段和活动，分析原因，给出一个具体改变方法。

### 六、三条可操作建议

给出 3 条明天就能开始执行的具体建议，分别对应：增加心流活动、调整时间安排、改善低效场景。

## 与上次分析对比

检查 `~/.flow-data/reviews/` 目录下是否有历史快照文件。

**如果有历史快照：**

读取最近一次快照，与本次分析对比，展示变化：

```
对比上次分析（X 月 X 日）：

心流率：X% → X%（↑X% / ↓X%）
平均专注度：X.X → X.X
心流活动排名变化：
  · [活动] 从第 X 名 → 第 X 名
高峰时段变化：[上次] → [本次]
```

如果变化明显，说明可能的原因（记录场景更丰富了、生活状态变化等）。

**如果没有历史快照：**

跳过对比，不做任何说明。

---

## 保存快照

将本次分析结果保存到 `~/.flow-data/reviews/` 目录，文件名为当天日期，例如 `2026-04-18.md`。

快照内容包括：分析日期、记录条数、心流率、平均专注度、高峰时段、心流活动排名前三。格式简洁，供下次对比用。

如果目录不存在，先创建目录再保存。

---

## 反思环节

分析和建议展示完后，问用户一个问题：

```
最后一个问题——

这次分析里，有没有让你意外的地方？
或者有什么结果，和你原来以为的不一样？
```

等用户回答。无论用户说什么，都不要评价对错，而是帮用户把这个感受和数据连起来，说一句有意义的话。比如：

- 用户说「没想到我在咖啡馆状态这么好」→ 「数据显示你在咖啡馆的心流率是 X%，比其他地方高出很多，这可能是一个值得刻意安排的环境。」
- 用户说「结果和我想的差不多」→ 「那说明你对自己的状态已经有比较清晰的感知了。接下来的挑战是把这个感知变成真正的行动改变。」

将用户的回答连同日期和本次核心数据，追加保存到 `~/.flow-data/reflections.md`：

```markdown
## 2026-04-18

**数据摘要：** X 条记录，心流率 X%，高峰时段 XX:XX-XX:XX

**我的感受：**
[用户原话]
```

---

## 结束语

```
本次分析完成。建议 3～6 周后再跑一次，观察你的变化。
生活状态变了，心流的来源也会跟着变。
```
